# Online_Shopping_System
<h2 style="font-size:18px;"> <b>LaSephora</b> is a feature-rich online shopping system built with HTML, Java, and MySQL. 
 <br><br>
 
 It streamlines cloth ordering, sales, and inventory management while providing robust tools for admin and supplier operations, user account management, and system maintenance.</h2>
 
<h2 style="font-size:22px;"><b>Features</b></h2>

<ul>
  
 <li> <b>Clothing Orders:</b> Simplifies the process of browsing and purchasing clothing items.</li>
  
  <li> <b>Data Management:</b>Efficiently handles user, order, and sales data for accuracy and easy retrieval.</li>
  
  <li><b>Sales Management: </b> Provides tools for tracking sales and generating insights.</li>
  
 <li><b>Secure User Authentication:</b> Features secure login, registration, and password reset functionalities to safeguard user data.</li>
  
 <li><b>Inventory Management: </b>Enables real-time stock tracking for admins and suppliers.</li>
  
 <li><b>Supply Management:</b> Optimizes supplier collaboration and order fulfillment.</li>

  
</ul>

<h2 style="font-size:22px;"><b>Database Management</b></h2>

The application uses MySQL within the MySQL Workbench server for efficient data handling and management, ensuring a reliable foundation for organizing and maintaining data effectively.

<b>Tables:</b>

<ul>

<li><b>Customer: </b>Stores customer details such as name, contact information, and account credentials.</li>
<li><b>Supplier: </b>Manages supplier information, including company details and product supply data.</li>
<li><b>Administrator: </b>Stores admin-related data.</li>
<li><b>newproduct: </b>Tracks information about newly added products, including descriptions, prices, and availability.</li>
<li><b>Orders: </b>Records customer orders, including product details, quantities, and order statuses.</li>
<li><b>Customer_Reviews:</b>Stores customer feedback and ratings for products and services.</li>

</ul>



<b>Triggers:</b> Auto-generation of primary keys for all tables ensures data integrity and consistency.
